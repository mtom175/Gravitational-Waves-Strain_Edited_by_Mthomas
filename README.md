Simulation Overview

The simulation models two black holes orbiting each other while gradually losing orbital energy through gravitational-wave emission.

As the black holes approach:

1. Orbital frequency increases.
2. Gravitational-wave amplitude grows.
3. A merger occurs.
4. The remnant black hole undergoes ringdown oscillations.

The generated signal is displayed in both time and frequency domains.
